# Mobile Apps

## Overview

Contains apps for Customers, Drivers, and Store Owners.

### Apps

- Customer App
- Driver App
- Store App

## Technologies

- Android (Java / Kotlin)
- Firebase (optional for notifications)

## Setup

1. Open Android Studio
2. Import the relevant module
3. Configure API endpoints in app configuration

## Notes

- Each app targets a specific user role.
- Firebase can be used for push notifications and messaging.
